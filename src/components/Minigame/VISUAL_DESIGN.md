## Recommended Real Sprite Style

### For Vampire Survivors-like Quality:

1. **32x32 or 64x64 pixels** per sprite
2. **Pixel art style** with limited colors (4-8 colors per sprite)
3. **Simple animation** (4 frames for walking is enough)
4. **Clear silhouettes** (should be recognizable from far away)
5. **Consistent style** across all sprites

### Example Character Sprite Sheet Layout:

```
Frame 0    Frame 1    Frame 2    Frame 3
[idle]     [walk1]    [walk2]    [walk3]
```

### Color Palette Suggestions:

**Player (Hero)**
- Main: Bright green or blue
- Accent: White/yellow
- Outline: Dark green/black

**Basic Enemy**
- Main: Red
- Accent: Dark red
- Eyes: Yellow/white

**Elite Enemy**
- Main: Purple/magenta
- Accent: Pink
- Glow: Bright pink (handled by code)

**Boss**
- Main: Orange/gold
- Accent: Yellow
- Size: 48x48 or 64x64 (larger than normal)

## Projectile Sprites

Projectiles can be smaller: **16x16 pixels**

- **Waterball**: Blue sphere with white highlight
- **Banana**: Yellow curved shape
- **Acorn**: Brown oval with cap
- **Boomerang**: Brown L-shape

## Effects

These are generated by code, no sprites needed:
- ✓ Blood splatters (red particles)
- ✓ Explosions (orange/yellow particles)
- ✓ Damage numbers (floating text)
- ✓ Charge glow (golden aura)
- ✓ Elite aura (purple pulsing glow)
